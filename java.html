Java Topics Reference

1. Java Introduction

Java is a platform-independent, object-oriented programming language developed by Sun Microsystems. It follows the principle of "Write Once, Run Anywhere".

Features: Simple, Secure, Platform Independent, Robust, Multithreaded, High Performance

public class Main {
  public static void main(String[] args) {
    System.out.println("Hello World");
  }
}
2. Variables and Data Types

Java variables must be declared with a type. Types include:

  • Primitive types: int, float, boolean, char, etc.
  • Non-primitive types: String, Arrays, Classes
int age = 25;
double price = 10.99;
boolean isJavaFun = true;
char grade = 'A';
String name = "John";
System.out.println(name + " is " + age + " years old.");
3. Operators

Java supports different types of operators:

  • Arithmetic: +, -, *, /, %
  • Relational: ==, !=, >, <, >=, <=
  • Logical: &&, ||, !
  • Assignment: =, +=, -=, etc.
int x = 10, y = 5;
System.out.println(x + y);
System.out.println(x > y);
System.out.println((x > y) && (y > 0));
4. Control Statements

Used to control the flow of execution:

  • Conditional: if, if-else, switch
  • Loops: for, while, do-while
  • Branching: break, continue, return
int number = 10;
if (number > 0) {
  System.out.println("Positive number");
} else {
  System.out.println("Negative number");
}

for (int i = 0; i < 5; i++) {
  if (i == 3) continue;
  System.out.println(i);
}
5. Methods

Methods are blocks of code that perform specific tasks. Java supports:

  • Static methods
  • Instance methods
  • Method overloading
public class Calculator {
  static int add(int a, int b) {
    return a + b;
  }

  int multiply(int a, int b) {
    return a * b;
  }

  public static void main(String[] args) {
    System.out.println(add(3, 4));
    Calculator c = new Calculator();
    System.out.println(c.multiply(2, 5));
  }
}
6. Classes and Objects

Classes are blueprints for objects. Objects are instances of classes.

class Student {
  String name;
  int age;

  void display() {
    System.out.println("Name: " + name);
    System.out.println("Age: " + age);
  }
}

public class Main {
  public static void main(String[] args) {
    Student s1 = new Student();
    s1.name = "John";
    s1.age = 20;
    s1.display();
  }
}
7. Inheritance

Inheritance allows reuse of code and promotes hierarchical classification.

class Animal {
  void sound() {
    System.out.println("Animal sound");
  }
}

class Dog extends Animal {
  void bark() {
    System.out.println("Dog barks");
  }
}

public class Test {
  public static void main(String[] args) {
    Dog d = new Dog();
    d.sound();
    d.bark();
  }
}
8. Arrays

Arrays are used to store multiple values in a single variable.

int[] numbers = {10, 20, 30, 40};
System.out.println("First element: " + numbers[0]);

for (int n : numbers) {
  System.out.println(n);
}
9. Strings

Strings are immutable objects that represent sequences of characters.

String s = "Hello";
System.out.println(s.length());
System.out.println(s.toUpperCase());
System.out.println(s.concat(" World"));
10. Exception Handling

Java provides robust error handling using try, catch, finally, and throw/throws.

public class Example {
  public static void main(String[] args) {
    try {
      int result = 10 / 0;
    } catch (ArithmeticException e) {
      System.out.println("Error: " + e.getMessage());
    } finally {
      System.out.println("This block always executes.");
    }
  }
}
