Python Topics Reference

1. Python Basics

Introduction: Python is a popular, beginner-friendly programming language known for its readability.

print("Hello, World!")

Syntax: Python uses indentation to define blocks of code instead of curly braces.

if True:
    print("Indented block")

Variables: Variables are created when you assign a value to a name.

name = "Alice"
age = 25
print(name, age)
2. Numbers and Operators

Numbers: Python has int, float, and complex number types.

x = 5       # int
y = 3.14    # float
z = 2 + 3j  # complex
print(type(x), type(y), type(z))

Operators: Used to perform operations like addition or comparison.

a = 10
b = 5
print(a + b, a > b, a == 10 and b == 5)
3. Control Flow

Conditionals: Execute blocks of code depending on conditions using if, elif, and else. 

age = 18
if age >= 18:
    print("Adult")
elif age >= 13:
    print("Teenager")
else:
    print("Child")

Nested conditionals: You can place conditionals inside conditionals.

x = 10
y = 5
if x > 5:
    if y > 2:
        print("x > 5 and y > 2")

Ternary operator: One-line conditional expression.

status = "Pass" if score >= 50 else "Fail"

match-case (Python 3.10+): Similar to switch-case in other languages.

def check_day(day):
    match day:
        case "Mon":
            print("Start of week")
        case "Fri":
            print("Weekend soon")
        case _:
            print("Midweek")

Loops: Repeating code using for or while.

# for loop
for i in range(3):
    print(i)

# while loop
count = 0
while count < 3:
    print(count)
    count += 1

break and continue: break exits loop, continue skips to next iteration.

for i in range(5):
    if i == 3:
        break
    print(i)

for i in range(5):
    if i == 2:
        continue
    print(i)

else with loops: Runs when loop completes normally (not broken).

for i in range(3):
    print(i)
else:
    print("Loop finished")

pass statement: Placeholder when no code is needed.

for i in range(3):
    pass  # To be implemented later
4. Data Collections

Lists: Ordered and changeable collections. Can contain duplicates.

fruits = ["apple", "banana"]
fruits.append("cherry")
fruits[1] = "orange"
print(fruits)

Tuples: Ordered and unchangeable. Use when data should not be modified.

colors = ("red", "green", "blue")
print(colors[0])

Sets: Unordered collections with unique elements. No duplicates allowed.

numbers = {1, 2, 3, 2}
numbers.add(4)
print(numbers)

Dictionaries: Key-value pairs for storing data. Keys must be unique.

student = {"name": "Alice", "age": 20}
student["age"] = 21
print(student["name"], student["age"])
5. Functions

Defining Functions: Functions are defined using the def keyword and are used to group reusable code.

def greet():
    print("Hello")

greet()

Function with Parameters: You can pass information into functions using parameters.

def greet(name):
    print("Hello,", name)

greet("Alice")

Default Parameters: Parameters can have default values.

def greet(name="Guest"):
    print("Hello,", name)

greet()
# Output: Hello, Guest

Return Statement: Functions can return values using return.

def add(a, b):
    return a + b

result = add(3, 4)
print(result)

Arbitrary Arguments: Use *args for variable number of positional arguments.

def total(*numbers):
    return sum(numbers)

print(total(1, 2, 3))

Keyword Arguments: Use **kwargs to accept keyword arguments.

def display(**info):
    for key, value in info.items():
        print(key, ":", value)

display(name="Alice", age=25)

Recursion: A function calling itself.

def factorial(n):
    if n == 1:
        return 1
    else:
        return n * factorial(n - 1)

print(factorial(5))

Lambda Functions: Anonymous functions written in a single line.

square = lambda x: x * x
print(square(5))

Function Annotations: Used for type hints.

def add(x: int, y: int) -> int:
    return x + y

print(add(3, 4))
6. File Handling

Reading/Writing Files: Files are opened with open() and should be closed after use. Using with handles closing automatically. 

# Writing to a file
with open("example.txt", "w") as f:
    f.write("Hello\nWorld")

# Reading the file
with open("example.txt", "r") as f:
    for line in f:
        print(line.strip())

Appending to a file: Use mode "a" to add to the end.

with open("example.txt", "a") as f:
    f.write("\nNew line")

Checking file existence (with os):

import os
if os.path.exists("example.txt"):
    os.remove("example.txt")
7. Object-Oriented Programming (OOP)

Classes and Objects: A class is a blueprint; an object is an instance of that blueprint.

class Person:
    def __init__(self, name):
        self.name = name

    def greet(self):
        return f"Hi, I'm {self.name}"

p = Person("John")
print(p.greet())

Inheritance: Allows new classes to inherit the properties of existing ones.

class Student(Person):
    def __init__(self, name, grade):
        super().__init__(name)
        self.grade = grade

s = Student("Alice", "A")
print(s.greet(), s.grade)

Encapsulation: Data hiding using private variables with _ or __. 

class BankAccount:
    def __init__(self):
        self.__balance = 0

    def deposit(self, amount):
        self.__balance += amount

    def get_balance(self):
        return self.__balance

Polymorphism: Same method name behaves differently based on context.

class Dog:
    def speak(self):
        return "Woof"

class Cat:
    def speak(self):
        return "Meow"

def animal_sound(animal):
    print(animal.speak())

animal_sound(Dog())
animal_sound(Cat())
11. Modules and Packages

Modules: A Python module is a file with a .py extension containing Python code (functions, variables, classes).

# Create a module file named mymodule.py
def greet(name):
    return f"Hello, {name}!"

# Use the module in another file
import mymodule
print(mymodule.greet("Alice"))

Importing Specific Elements: You can import specific functions or variables.

from mymodule import greet
print(greet("Bob"))

Using Aliases: Use as to rename imports.

import math as m
print(m.sqrt(25))

Packages: A package is a collection of modules in a directory with an __init__.py file.

# Directory structure:
# mypackage/
# ├── __init__.py
# ├── module1.py
# └── module2.py

from mypackage import module1
module1.some_function()

Built-in Modules: Python comes with standard libraries like math, os, sys, random.

import random
print(random.randint(1, 10))

Installing External Packages: Use pip to install third-party packages.

pip install requests

Using Installed Package:

import requests
response = requests.get("https://example.com")
print(response.status_code)
9. Date and Time

datetime Module: Used to work with dates and times.

import datetime
now = datetime.datetime.now()
print("Current Date and Time:", now)
print("Year:", now.year)
print("Month:", now.month)
print("Day:", now.day)

Create Specific Date:

d = datetime.datetime(2025, 7, 7)
print(d.strftime("%A, %d %B %Y"))

Time Delta: For date/time arithmetic.

from datetime import timedelta
tomorrow = now + timedelta(days=1)
print("Tomorrow:", tomorrow)
10. Math, JSON, Regex

Math Module: Provides mathematical functions.

import math
print("Square root:", math.sqrt(16))
print("Power:", math.pow(2, 3))
print("Pi:", math.pi)

JSON Module: For encoding/decoding JSON data.

import json
# Convert dictionary to JSON string
data = {"name": "Alice", "age": 25}
json_str = json.dumps(data)
print(json_str)

# Convert JSON string to dictionary
parsed = json.loads(json_str)
print(parsed["name"])

Regular Expressions (Regex): For pattern matching and searching in strings.

import re
text = "My email is test@example.com"
match = re.search(r"[\w.-]+@[\w.-]+", text)
if match:
    print("Found email:", match.group())

# Find all matches
text = "cat, mat, bat"
matches = re.findall(r".at", text)
print(matches)
11. Error and Exception Handling

try-except: Used to catch and handle exceptions.

try:
    print(10 / 0)
except ZeroDivisionError:
    print("Can't divide by zero")

Multiple exceptions: Handle different exceptions separately.

try:
    num = int("abc")
except ValueError:
    print("Invalid value")
except Exception as e:
    print("Error:", e)

finally block: Executes no matter what.

try:
    print("Trying...")
finally:
    print("Always runs")

Raising exceptions: You can manually throw an exception using raise.

def set_age(age):
    if age < 0:
        raise ValueError("Age cannot be negative")
    print("Age set to", age)
12. Advanced Topics

Decorators: Modify function behavior.

def decorator(func):
    def wrapper():
        print("Before")
        func()
        print("After")
    return wrapper

@decorator
def say_hello():
    print("Hello")

say_hello()
